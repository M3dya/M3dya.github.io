# Medya Asgharian — UX Research & Service Design Portfolio

Personal portfolio of Medya Asgharian, a UX Researcher and Service Designer specialising in data-driven workflows, platform experiences, and civic service design. This repo is the source for a static HTML/CSS site hosted on GitHub Pages.

**Live site:** https://m3dya.github.io

---

## Structure

```
├── index.html          # Home — hero and project card grid
├── about.html          # Background, experience, skills, and education
├── finch3d.html        # Case study: Finch 3D onboarding redesign
└── images/
    └── finch/          # Assets for the Finch 3D case study
```

## Featured Case Studies

- **Finch 3D** — Diagnosed an onboarding failure where domain jargon blocked first-time users, then designed a 4-step guided flow that cut projected onboarding time by ~20%.
- **Glocal Foundation** — IA and UX redesign for a civic non-profit, including a tax information tool and a Canadian federal elections data visualisation website.
- **Ontario Daycare Access** — Service design research across a 14-week sprint: bilingual surveys, journey maps, and service blueprints surfacing systemic barriers to childcare access in Ontario.
- **IdeateLive AI Platform** — End-to-end mixed-methods research for a B2B AI platform; 17+ user interviews informing shipped onboarding and AI interaction improvements.
